# Original Data With Chunk And Edge

이 디렉터리의 DOCX 파일은 원본 법령 데이터가 아니다.

각 DOCX 파일은 `original-data-toon/`의 원본 TOON 법령 데이터를 바탕으로 사람이 정리한 청크/엣지 설계 문서다. 문서 안에는 다음 정보가 들어 있다.

- 어떤 조문 또는 조문 묶음을 하나의 청크로 볼지
- 어떤 청크와 어떤 청크가 연결되어야 하는지
- 연결 이유, 관계 유형, 고령층 관점의 활용 포인트
- GraphRAG 비교 실험에서 확인해야 할 약한 연관 또는 직접 연관

예를 들어 아래 형태의 라인은 청크 간 edge를 의미한다.

```text
002_제2조_정의 -> 011_제11조_적용범위
017_제17조_근로조건의명시 -> 019_제19조_근로조건의위반
```

`original-data-toon/`은 실제 원문 데이터이고, 이 디렉터리의 DOCX는 그 원문을 어떻게 청킹하고 edge로 연결할지 설명하는 별도 입력 자료다. Neo4j 그래프를 만들 때는 이 DOCX 파일들을 먼저 읽고, DOCX에 적힌 청크/엣지 설계를 기준으로 그래프를 구성해야 한다.
