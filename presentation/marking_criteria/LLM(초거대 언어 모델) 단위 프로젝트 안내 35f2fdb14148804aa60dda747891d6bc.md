# 프로젝트 목표

- 환각을 방지하고 원하는 내외부 데이터 내에서 RAG 기반 LLM 활용 질의 응답 시스템 구현
- 문서를 벡터 형태로 임베딩하여 벡터데이터베이스에 저장 및 검색
- LangChain을 활용하여 벡터데이터베이스와 LLM 연동

# 세부 수행 절차 및 내용

- 데이터 수집 및 가공
- 벡터 데이터베이스 생성 및 데이터 저장
- One-shot 또는 Few-shot 활용 프롬프트 템플릿 작성
- 사용할 LLM 인공지능 모델 선택
- LangChain 활용 RAG 기술로 벡터데이터베이스와 LLM 연동하여 질의응답 구현
- 구현 결과 테스트 및 개선

# 주요 산출물

- 수집된 데이터 및 데이터 전처리 문서
- 시스템 아키텍처
- 개발된 소프트웨어: RAG 기반 LLM과 벡터데이터베이스 연동 구현 코드
- 테스트 계획 및 결과 보고서
