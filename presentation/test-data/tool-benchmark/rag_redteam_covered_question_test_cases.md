# RAG Red Team Covered Question Test Cases

## Selection 기준

- 기준 RAG 그래프: `rag-red-team` Neo4j에 적재한 4개 법령 문서만 사용.
- 포함 문서: `cleaned_근로기준법_raw.json.md`, `cleaned_고용보험법_raw.json.md`, `cleaned_근로자퇴직급여 보장법_raw.json.md`, `cleaned_고용상 연령차별금지 및 고령자고용촉진에 관한 법률_raw.json.md`.
- 제외 문서: 최저임금법, 기초연금법, 노인복지법, 조례, 지역 CSV, KOSIS, 잡코리아, 노인일자리/노인맞춤돌봄 PDF 등 현재 그래프에 없는 자료.
- 혼합 질문은 질문 전체를 위 4개 문서만으로 답할 수 있을 때만 포함했다. 일부 키워드만 겹치지만 외부 문서가 필요한 질문은 제외했다.
